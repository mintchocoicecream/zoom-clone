# Noom

Zoom Clone using NodeJS, WebRTC and Websockets.
#### 실시간 화상 채팅 웹 서비스 만들기 프로젝트
----
### - Skill
  - NodeJS, SocketIO, WebRTC
### - Requirements
  - Nodemon    
      ``` npm i nodemon -D ```
  - Babel    
      ``` npm i @babel/core @babel/cli @babel/node @babel/preset-env -D ```
  - express    
      ``` npm i express ```
  - pug    
      ``` npm i pug ``` 
  - webSocket    
      ``` npm i ws ``` 
  - SocketIO   
      ``` npm i socket.io ```
